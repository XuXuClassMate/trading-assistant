    🔧 Installation / 安装¶

    Docker (Recommended) / 推荐¶

    # Pull from GitHub Container Registry
docker pull ghcr.io/xuxuclassmate/trading-assistant:latest

# Quick start
mkdir -p trading-assistant-config && cd trading-assistant-config
curl -O https://raw.githubusercontent.com/XuXuClassMate/trading-assistant/main/.env.example
cp .env.example .env
# Edit .env with your API keys

docker run --rm -it \
  -v $(pwd)/.env:/app/.env \
  -v $(pwd)/watchlist.txt:/app/watchlist.txt \
  ghcr.io/xuxuclassmate/trading-assistant:latest \
  --help

    GitHub Releases¶

    pip install https://github.com/XuXuClassMate/trading-assistant/releases/download/v1.2.0/openclaw_trading_assistant-1.2.0-py3-none-any.whl

    🔐 Required Secrets / 需要的密钥¶

    GitHub Repository Secrets¶

    Secret Purpose Status
    GITHUB_TOKEN Auto-generated, for GitHub Packages ✅ Configured
    DOCKERHUB_USERNAME Docker Hub username ⏳ Optional
    DOCKERHUB_TOKEN Docker Hub access token ⏳ Optional

    To Enable Docker Hub¶

    1. Go to https://hub.docker.com/settings/security
    2. Create access token
    3. Add to repository secrets:
    4. DOCKERHUB_USERNAME: your Docker Hub username
    5. DOCKERHUB_TOKEN: your access token
