## 🔒 新安全模型

### 本技能做什么:
- ✅ **仅**从标准环境变量读取 API 密钥
- ✅ **仅**读取自身目录中的 `.env` 文件
- ✅ 向 **Twelve Data** 和 **Alpha Vantage** API 发送 HTTP 请求
- ✅ 将日志和报告写入**自身目录**（`logs/`, `reports/`, `data/`）
- ✅ 读取用户配置的 `watchlist.txt` 和 `config.json`

### 本技能**不**做什么:
- ❌ **不**读取父目录的 `.env` 文件
- ❌ **不**导入外部包
- ❌ **不**访问项目目录外的文件
- ❌ **不**向未授权的端点发送数据
- ❌ **不**执行后台任务
- ❌ **不**需要券商凭证

```bash

# 安装依赖
```
