# 持仓成本价显示问题修复

## 问题诊断

### 现象
日报中持仓成本价显示为 `$0.00`：
```
| 股票 | 数量 | 成本 | 现价 | 市值 | 盈亏 | 占比 |
|------|------|------|------|------|------|------|
| IBKR | 100 | $0.00 | $0.00 | $6,957.00 | +10.90 | 63.3% |
```

### 原因
1. **非交易时间** - API 无法获取实时价格，`current_price` 为 0
2. **数据源问题** - Twelve Data 在美股休市期间返回 0 或前一交易日收盘价

### 实际数据 (holdings.json)
```json
{
  "symbol": "IBKR",
  "shares": 100,
  "avg_cost": 69.46,  ← 成本价正确
  "current_price": 0,  ← 非交易时间为 0
  "market_value": 6957,
  "cost_basis": 6946,
  "unrealized_pnl": 10.90
}
```

---

## 修复方案

### 方案 1: 使用 cost_basis 计算 (已实施)

```python
for h in portfolio.get("holdings", []):
    if h.get("shares", 0) > 0:
        # 使用 avg_cost 显示成本
        cost = h.get('avg_cost', 0)
        
        # 如果 current_price 为 0，从 market_value 估算
        price = h.get('current_price', 0)
        if price == 0 and cost > 0:
            price = h.get('market_value', 0) / h.get('shares', 1)
        
        port_section += f"| {h['symbol']} | {h['shares']:,.0f} | ${cost:.2f} | ${price:.2f} | ..."
```
