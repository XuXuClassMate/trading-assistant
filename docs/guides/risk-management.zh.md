# ⚠️ 风险管理

**版本**: v1.5.0  
**最后更新**: 2026-03-25

---

## 📊 概述

风险管理是交易成功的关键，帮助你在追求收益的同时控制潜在损失。

### 核心原则

- ✅ **保本第一** - 永远不要冒失去本金的风险
- ✅ **风险收益比** - 每笔交易至少 1:2 的风险收益比
- ✅ **分散投资** - 不要把所有鸡蛋放在一个篮子里
- ✅ **止损纪律** - 严格执行止损，不抱侥幸心理

---

## 🎯 风险类型

### 1. 市场风险

**定义**: 市场价格波动导致的损失风险

**管理方法**:
- 设置止损
- 控制仓位
- 分散投资
- 对冲策略

### 2. 流动性风险

**定义**: 无法及时平仓或滑点过大的风险

**管理方法**:
- 选择高流动性标的
- 避免大额一次性交易
- 使用限价单而非市价单

### 3. 操作风险

**定义**: 人为错误或系统故障导致的风险

**管理方法**:
- 制定交易计划
- 双重确认重要操作
- 定期备份数据
- 使用可靠的交易平台

### 4. 情绪风险

**定义**: 情绪化决策导致的风险

**管理方法**:
- 严格执行交易计划
- 设置自动止损
- 定期休息
- 记录交易日志

## 📈 风险管理工具

### 1. 止损 (Stop Loss)

**定义**: 预先设定的最大可接受损失

**设置方法**:

| 方法 | 公式 | 示例 |
|------|------|------|
| 百分比止损 | 入场价 × (1 - 止损%) | $100 × 0.95 = $95 |
| 技术位止损 | 支撑位 - 缓冲 | $98 - $2 = $96 |
| ATR 止损 | 入场价 - (2 × ATR) | $100 - (2 × $3) = $94 |

### 2. 止盈 (Take Profit)

**定义**: 预先设定的获利了结价格

**设置方法**:

| 方法 | 公式 | 示例 |
|------|------|------|
| 固定比例 | 入场价 × (1 + 目标%) | $100 × 1.10 = $110 |
| 风险收益比 | 入场价 + (风险 × 倍数) | $100 + ($5 × 2) = $110 |
| 技术位 | 阻力位 - 缓冲 | $112 - $2 = $110 |

### 3. 仓位控制

**原则**: 单笔风险不超过总资金的 2%

**公式**:
```
仓位大小 = (总资金 × 风险%) / (入场价 - 止损价)
```

**示例**:
- 总资金：$10,000
- 风险：2% ($200)
- 入场：$100
- 止损：$95
- 仓位：$200 / $5 = 40 股

## 🛠️ 实用技巧

### 1. 风险收益比计算

**公式**:
```
风险收益比 = (目标价 - 入场价) / (入场价 - 止损价)
```

**标准**:
- 最低：1:2 (赚 2 赔 1)
- 理想：1:3 (赚 3 赔 1)
- 优秀：1:4+ (赚 4 赔 1)

**示例**:
- 入场：$100
- 止损：$95 (风险 $5)
- 目标：$110 (收益 $10)
- 风险收益比：10/5 = 1:2 ✅

### 2. 最大回撤控制

**定义**: 账户从最高点下跌的最大幅度

**控制方法**:
- 设置每日最大损失 (如 5%)
- 设置每周最大损失 (如 10%)
- 设置每月最大损失 (如 20%)
- 触及限制立即停止交易

### 3. 相关性管理

**问题**: 持有高度相关的多个仓位会增加风险

**解决**:
- 检查仓位相关性
- 避免同时持有高度相关标的
- 分散到不同行业/市场

## ⚠️ 常见错误

### 1. 不设止损

**错误**: "价格会回来的"

**后果**: 小亏变大亏，甚至爆仓

**纠正**: 每笔交易必须设止损

### 2. 移动止损过紧

**错误**: 稍有波动就被止损

**后果**: 频繁止损，错过大趋势

**纠正**: 给予合理波动空间

### 3. 亏损加仓

**错误**: "摊平成本"

**后果**: 越陷越深，损失放大

**纠正**: 只在盈利时加仓

### 4. 情绪化交易

**错误**: 报复性交易、FOMO

**后果**: 连续亏损，失去理智

**纠正**: 暂停交易，调整心态

## 📊 风险检查清单

### 交易前

- [ ] 这笔交易的风险收益比是多少？
- [ ] 止损位设置在哪里？
- [ ] 仓位大小是否合适？
- [ ] 最坏情况会损失多少？
- [ ] 这个损失我能接受吗？

### 交易中

- [ ] 止损是否已设置？
- [ ] 是否需要调整止损？
- [ ] 市场条件是否改变？
- [ ] 是否按计划执行？

### 交易后

- [ ] 这笔交易结果如何？
- [ ] 是否严格执行了计划？
- [ ] 有什么可以改进的？
- [ ] 从中学到了什么？
