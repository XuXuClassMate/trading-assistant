# 🛠️ TradingAgents 工具

## AI 分析师

- **Market Analyst** - 市场趋势分析
- **News Analyst** - 新闻情绪分析
- **Social Media Analyst** - 社交媒体情绪
- **Fundamentals Analyst** - 财务指标

## AI 智能体

- **Bull Researcher** - 看涨研究
- **Bear Researcher** - 看跌研究
- **Portfolio Manager** - 仓位管理
- **Trader** - 交易执行

## 风险管理

- **Aggressive Debator** - 高风险/高回报
- **Conservative Debator** - 低风险
- **Neutral Debator** - 平衡观点

## 数据源

- **Twelve Data** - 价格数据
- **Alpha Vantage** - 指标和新闻
- **Binance** - 加密货币数据

## CLI 命令

```bash
ta analyze NVDA
ta monitor
ta backtest --days 30
ta verify
```
