## 📊 概述

技术分析是交易助手的核心功能，通过多种算法计算支撑位和阻力位，帮助识别关键价格水平。

### 核心特性

## 🎯 支撑位与阻力位

### 支撑位 (Support)

**定义**: 价格下跌时可能遇到支撑并反弹的价格水平

**形成原因**:
- 前期低点
- 成交量密集区
- 心理整数关口

### 阻力位 (Resistance)

**定义**: 价格上涨时可能遇到阻力并回落的价格水平

**形成原因**:
- 前期高点
- 套牢盘密集区
- 心理整数关口

---

## 📈 计算方法

### 1. 前期高低点法

**原理**: 历史高低点往往成为未来的支撑/阻力

```python
# 简化示例
support = min(recent_lows)
resistance = max(recent_highs)
```

### 2. 移动平均线法

**原理**: 均线作为动态支撑/阻力

**常用周期**:
- MA20 - 短期
- MA50 - 中期
- MA200 - 长期

### 3. 斐波那契回撤

**原理**: 市场往往在斐波那契比例位置反转

**关键水平**:
- 23.6%
- 38.2%
- 50.0%
- 61.8%
- 78.6%

## 📊 实际应用

### 交易策略

#### 支撑位买入

**条件**:
- 价格接近支撑位
- 出现反弹信号 (RSI 超卖等)
- 成交量放大

**操作**:
- 在支撑位附近买入
- 止损设在支撑位下方 2-3%
- 目标价：下一个阻力位

#### 阻力位卖出

**条件**:
- 价格接近阻力位
- 出现回落信号 (RSI 超买等)
- 成交量萎缩

**操作**:
- 在阻力位附近卖出
- 止损设在阻力位上方 2-3%
- 目标价：下一个支撑位

### 突破交易

**支撑位突破**:
- 价格跌破支撑 → 可能继续下跌
- 等待回踩确认
- 顺势做空

**阻力位突破**:
- 价格突破阻力 → 可能继续上涨
- 等待回踩确认
- 顺势做多

## ⚠️ 注意事项

### 局限性

1. **主观性**: 不同方法可能得出不同结果
2. **滞后性**: 基于历史数据
3. **假突破**: 价格可能短暂突破后反转
4. **市场环境**: 不同市场状态效果不同

### 最佳实践

1. **多方法确认**: 使用多种计算方法交叉验证
2. **结合其他指标**: 配合 RSI、MACD 等使用
3. **设置止损**: 始终设置止损保护
4. **关注成交量**: 突破时成交量确认
