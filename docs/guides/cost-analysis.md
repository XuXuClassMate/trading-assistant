## Overview

Position cost analysis helps you understand your trading costs and optimize position sizing.

## Features

- Calculate average entry price
- Track position P&L
- Analyze cost basis

## Usage

```bash
ta pos --symbol NVDA --price 175 --capital 10000 --risk 2
```
